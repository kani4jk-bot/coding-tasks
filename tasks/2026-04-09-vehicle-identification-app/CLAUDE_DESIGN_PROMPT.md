# Claude Design Prompt — Tesla-style UI

Use `DESIGN.md` as the primary visual/system brief for this project.

When designing or editing UI for this vehicle identification app:
- follow the Tesla-inspired principles in `DESIGN.md` first
- favor a minimal, premium, engineering-led interface with restrained color, strong hierarchy, and a sleek mobile-first feel
- keep the capture flow direct, the vehicle image prominent, and the result view focused on core identification details before secondary facts/history
- prefer clean panels, sparse controls, simple navigation, and crisp typography over decorative automotive clichés
- preserve accessibility, implementation realism, and native-feeling behavior for the existing FastAPI + Expo / React Native stack

For every UI task:
1. briefly state which Tesla design cues you are applying
2. keep layouts minimal, high-signal, and performance-minded
3. avoid drifting into flashy racing aesthetics, cluttered dashboards, or overly playful styling unless explicitly asked
