# coding-tasks

Repository for coding tasks completed through OpenClaw.

## Structure

- `tasks/` — individual coding task folders

## Suggested task folder format

Each completed task can live in its own folder, for example:

- `tasks/2026-03-25-build-landing-page/`
- `tasks/2026-03-25-fix-login-bug/`

Inside each task folder, I can include:

- source code
- notes
- setup instructions
- output artifacts

## Workflow

For future coding tasks, the default flow is:
1. Create a task folder in `tasks/`
2. Complete the work
3. Commit the result
4. Push to GitHub
